MATLAB CVIP Toolbox Functions

Compression

The compression can be applied in any image. The compression function includes lossy and lossless compression.

  • kmeans_cvip -do k-means compression
  • kmeans_plus_cvip -do k-means++ compression
  • mBTCenco_cvip -do a multi-level block truncation coding compression algorithm with four different level selecitons
  • bRLCenco__cvip -do a bitplane run-length coding compression algorithm
