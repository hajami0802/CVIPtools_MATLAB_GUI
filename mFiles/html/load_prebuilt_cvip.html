LOAD_PREBUILT_CVIP

load_prebuilt_cvip() - loads all of the current MATLAB prebuilt demo image.

Contents

  • SYNTAX
  • DESCRIPTION
  • REFERENCE
  • EXAMPLE
  • CREDITS

SYNTAX

load_prebuilt_cvip

Output Parameters include:

  • imshow(figure)

DESCRIPTION

This function loads all of the current MATLAB prebuilt demo images. Loads them into the workspace with the name of the image as the variable name.

REFERENCE

1. Scott E Umbaugh. DIGITAL IMAGE PROCESSING AND ANALYSIS: Applications with MATLAB and CVIPtools, 3rd Edition.

EXAMPLE

 

% Calling function

load_prebuilt_cvip;

% Display output image

figure;imshow(ligthouse);title('Output  image');
