ROTMASK_CVIP

rotmask_cvip() - Rotate a 2-D mask or filter by given angle ( in degrees).

Contents

  • SYNTAX
  • DESCRIPTION
  • REFERENCE
  • EXAMPLE
  • CREDITS

SYNTAX

outMask = rotmask_cvip(inMask, angleD)

Input Parameters include :

  • inMask - A mask or filter of NxN size.
  • angle - An angle in degree.

Output Parameter includes :

  • outMask - Rotated mask or filter.

DESCRIPTION

The function rotates an 2-D array by given angle (in degrees). This function is useful to rotate the mask used in the thinning operation.

REFERENCE

1. Scott E Umbaugh. DIGITAL IMAGE PROCESSING AND ANALYSIS: Applications with MATLAB and CVIPtools, 3rd Edition.

EXAMPLE

% Mask filter

  M = [0 0 0; nan 1 nan; 1 1 1];

% angle

 angle = 45;

% Calling function

 O = rotmask_cvip(M,angle)

O =

   NaN     1     0
     1     1   NaN
     1     1   NaN
