SPLIT_MERGE_CVIP

split_merge_cvip() - Performs Split-Merge Segmentation of an image.

Contents

  • SYNTAX
  • DESCRIPTION
  • REFERENCE
  • EXAMPLE
  • CREDITS

SYNTAX

[outImage] = split_merge_cvip( inImage, N, testType, testArgs)

Input Parameters include:

  • inImage - 2-D or 3-D Image array. The image can be of uint8, int16, int32, single or double class.
  • N - Initial level of split in each direction (horizontal & vertical). After initial splitting, (N+1)^2 subregions are formed. (default| 7)
  • testType - Type of the homogeneity test (default| 2)
  • testArgs - Parameter of homogeneity test. Only Variance test has the input parameter (i.e. threshold value). For more details, refer to Homogeneity Test section.

Output Parameter includes:

  • outImage - Segmented image using Split and Merge method

DESCRIPTION

The function performs Split & Merge Segmentation on a grayscale image. If input image is 3-band image, it performs PCT transform and then segments only the principal component. To perform either split or merge operation, homogeneity test is required. A homogeneity test is used to determine if the Region of Interest is homogeneous, if it is then all the pixels in that region are replaced with their average (in the segmented image). The level to begin the split and merge is related to the size of the beginning region, specifically, the smaller the number the larger the beginning region. The chosen homogeneity test is the test that is to be used throughout the split and merge segmentation process.

REFERENCE

1. Scott E Umbaugh. DIGITAL IMAGE PROCESSING AND ANALYSIS: Applications with MATLAB and CVIPtools, 3rd Edition.

EXAMPLE

% Read Image

 I = imread('cam.bmp');

% calling function with default parameters

 O1 = split_merge_cvip(I);

% segemented image with non-default input parameters

%N=7, testType = 3 (variance test), testArgs = 20 (threshold)

 O2 = split_merge_cvip(I,7,3,15);

% Display input image

 figure;imshow(I);title('Input Image');

% Display output image

 figure;imshow(O1,[]);title('Output Image with default parameters');

 figure;imshow(O2,[]);title('Output Image with user specified parameters');
