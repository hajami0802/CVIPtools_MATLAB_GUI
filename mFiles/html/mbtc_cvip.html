MBTC_CVIP

mBTCenco_cvip() - performs multi-level block truncation coding compression with four different level selecitons.

mBTCenco_cvip() - performs multi-level block truncation coding and decompress the data file.

  • SYNTAX
  • DESCRIPTION
  • REFERENCE
  • EXAMPLE
  • CREDITS

SYNTAX

fileData = mBTCenco_cvip(InIma,Block,Level)
OutIma = mBTCdeco_cvip(fileData)

Input Parameters include:

  • InIma - Input image can be gray image or rgb image of MxN size.
  • Block - Size of a block (Block*Block area)
  • Level - Number of pixel values allowed after compression per block

Output Parameters include:

  • fileData- compressed image data
  • OutIma- the output image

DESCRIPTION

This function performs a multi-level block truncation coding compression algorithm with four different level selecitons

REFERENCE

1. Scott E Umbaugh. DIGITAL IMAGE PROCESSING AND ANALYSIS: Applications with MATLAB and CVIPtools, 3rd Edition.
